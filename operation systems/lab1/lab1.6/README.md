1. Разработать собственную функцию для создания ядерных потоков — аналог `pthread_create()`:
```C
int mythread_create(mythread_t thread, void *(start_routine), void *arg);
```
Функция должна возвращать успех-неуспех.

a. `mythread_join` должен быть. Любого уровня примитивности.
b. `mythread_create` при ошибке в каком-то из внутренних системных вызовов
должен сообщать пользователю, что что-то пошло не так, а не просто успешно
отрабатывать (по аналогии с `pthread_create`, опять же).
c. Тестирование вашей реализации должно быть сложнее, чем create + join.
Создавайте несколько потоков, проверяйте ошибки - пока хотя бы для себя не
сможете сказать, что реализация протестирована.
d. `mythread_cancel` писать не нужно, но как его реализовали бы - быть готовым
ответить надо.
